Анализ данных пользователей и A/B-тестирования для интернет-магазина BitMotion Kit
Описание проекта
В данном проекте я провел анализ двух различных наборов данных:

Анализ активности пользователей из Москвы и Санкт-Петербурга:

Исследовал гипотезу о различии средней активности пользователей между двумя городами

Использовал статистические методы (t-тест) для проверки гипотезы

Проанализировал распределение пользователей и их активность

Оценка результатов A/B-теста для интернет-магазина BitMotion Kit:

Проверил корректность проведения теста (равномерность распределения, отсутствие пересечений)

Проанализировал результаты тестирования нового интерфейса сайта

Оценил влияние изменений на поведение пользователей

Основные цели исследования:

Проверить статистические гипотезы о различиях в поведении пользователей

Оценить эффективность изменений интерфейса интернет-магазина

Подготовить рекомендации для бизнеса на основе данных

Использованные библиотеки:

Pandas для работы с данными

Matplotlib и Seaborn для визуализации

Scipy для статистического анализа

Statsmodels для дополнительных статистических тестов

Выводы
По анализу активности пользователей:

Не удалось обнаружить статистически значимых различий в средней активности между пользователями Москвы и Санкт-Петербурга (p-value = 0.211)

Распределение количества часов активности в двух городах практически одинаковое

Возможная причина отсутствия различий - схожий ритм жизни и свободного времени в этих городах

По A/B-тестированию:

Распределение пользователей между группами A и B практически равномерное (разница всего 1.06%)

Тест был проведен корректно, без пересечений с другими тестами

Для окончательных выводов о результатах теста необходимо провести дополнительный анализ поведения пользователей и их конверсии

Рекомендации:

Для интернет-магазина BitMotion Kit следует продолжить анализ A/B-теста с фокусом на ключевые метрики (конверсия, средний чек)

Можно рассмотреть другие гипотезы для улучшения пользовательского опыта, так как текущие изменения интерфейса не показали значимого влияния на активность пользователей

Для более глубокого понимания различий между городами можно провести дополнительное исследование с учетом других факторов (возраст, пол, интересы пользователей)
