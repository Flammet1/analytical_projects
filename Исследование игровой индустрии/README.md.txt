Исследование развития игровой индустрии в начале XXI века

Описание проекта:

В данном проекте был проведен анализ данных о продажах компьютерных игр, их оценках пользователями и критиками, а также других характеристик за период с 2000 по 2013 год. Основные цели исследования включали:

Предобработку данных: проверку на пропуски, дубликаты и корректность типов данных.

Фильтрацию данных по заданному временному периоду.

Категоризацию игр по оценкам пользователей и критиков.

Выявление топ-7 платформ по количеству выпущенных игр.

Проведение анализа и формулирование выводов о развитии игровой индустрии.

Для выполнения проекта были использованы следующие библиотеки Python:

Pandas — для работы с данными, их обработки и анализа.

NumPy — для выполнения численных операций.

Выводы:

Предобработка данных:

Были обнаружены и обработаны пропуски в данных, особенно в столбцах с оценками (critic_score, user_score, rating). Пропуски заполнены медианными значениями или заменены на индикаторы (например, -1 для числовых данных и "нет данных" для категориальных).

Удалены дубликаты, что составило 1.41% от общего объема данных.

Приведены к единому формату названия столбцов и категориальные данные (жанры, платформы, рейтинги).

Фильтрация данных:

Данные были отфильтрованы по периоду с 2000 по 2013 год, что позволило сосредоточиться на актуальном для исследования временном отрезке.

Категоризация оценок:

Игры были разделены на три категории по оценкам пользователей и критиков: высокая, средняя и низкая. Это позволило выделить наиболее успешные игры и платформы.

Топ-7 платформ:

Были определены платформы с наибольшим количеством выпущенных игр за исследуемый период. Это дало представление о популярности платформ среди разработчиков.

Общие тенденции:

Анализ показал, что игры с высокими оценками пользователей и критиков чаще выпускались на определенных платформах.

Были выявлены жанры, которые пользовались наибольшей популярностью в начале XXI века.

Итогом работы стала подготовка данных для дальнейшего анализа, который может быть использован для прогнозирования тенденций в игровой индустрии и принятия решений о разработке новых игр.
